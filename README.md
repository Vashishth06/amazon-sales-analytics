# Amazon Sales Analysis

Sales analytics for Amazon marketplace using Azure Databricks and PySpark.

## Dataset

- **Records:** 50,000 orders
- **Date Range:** 2022 - 2024
- **Categories:** Books, Fashion, Sports, Beauty, Electronics, Home & Kitchen
- **Regions:** North America, Asia, Europe, Middle East

## Notebooks

1. **01_data_loading_and_cleaning.ipynb** - Data ingestion, quality checks, feature engineering
2. **02_exploratory_analysis.ipynb** - Statistical analysis, top performers by category and region
3. **03_visualizations.ipynb** - Revenue trends, category performance, regional analysis

## Key Insights

- Total Revenue: Analyzed across product categories and regions
- Top Payment Method: UPI and Credit Card transactions
- Seasonal Trends: Quarterly revenue patterns identified
- Regional Performance: Revenue distribution across 4 major regions

## Tech Stack

- Azure Databricks
- PySpark
- Delta Lake
- Unity Catalog

## Setup

Run notebooks sequentially in Databricks environment.
